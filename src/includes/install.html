In order to install UpCom run the following commands in your console:


$ wget -qO - http://packages.updroid.com/Release.key | sudo apt-key add -

$ sudo sh -c 'echo "deb http://packages.updroid.com/ubuntu/ trusty main" > /etc/apt/sources.list.d/updroid.list'

$ sudo apt-get update

$ sudo apt-get install cmdr

In order to update run:


# Expect updates to be released weekly, more or less.
$ sudo apt-get update
$ sudo apt-get install --only-upgrade cmdr

Usage


# Set the ROS source folder
$ . /opt/ros/indigo/setup.bash
# Start UpCom
$ cmdr gui

To access UpCom: tune browser to localhost:12060


# Note: Raspberry Pi Users need to install ffmpeg >= 2.6.2 before using UpCom

# Because UpCom is still in alpha, it currently doesn't have any built in security tools. As such, please run UpCom from within a closed network.
