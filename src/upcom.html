UpDroid Commander (UpCom) is a customizable browser-based IDE designed to streamline robotics software development.

Workflow

By default UpCom will create and monitor a folder called UpRoot on first launch. Open and modify files on the fly

Control

No fumbling around with multiple terminals. Start any launch file or node from within one simple menu. One of the primary advantages of UpCom is consolidating all the tools you need to streamline the development process.

Console

The UpCom console is a projection of your own terminal window into the browser. Having a console integrated into the UpCom IDE means less switching between windows during development.

Tidy

Have more than one project? Easily switch between them and create new ones on the fly.
