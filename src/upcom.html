UpDroid Commander (UpCom) is a browser-based IDE for robotics software development, a GUI for the ROS (Robot Operating System) framework.

It can be installed on an Ubuntu 14.04 x86_64 PC that is connected to robotics hardware (sensors, actuators) alongside ROS.

UpCom is comprised of three core components designed to expedite robotics software development: file explorer, text editor, and console.

On top of this there are specialized tabs such as camera and teleop which you can install on a whim. Soon you will be able to add user created tabs as well.

Workflow

By default, UpCom will watch a folder called UpRoot which is created on its first launch unless you specify another workspace path. The explorer will show the file system which can be manipulated from within the browser.

Control

No fumbling around with multiple terminals. Start any launch file or node from within one simple menu. One of the primary advantages of UpCom is consolidating all the tools you need to streamline the development process.

Console

The UpCom console is a projection of your own terminal window into the browser. Having a console integrated into the UpCom IDE means less switching between windows during development.

Tidy

Have more than one project? Easily switch between them and create new ones on the fly.
