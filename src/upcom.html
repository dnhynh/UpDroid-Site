UpDroid Commander (UpCom) is a browser-based IDE for robotics software development.

It can be installed on an Ubuntu 14.04 x86_64 PC that is connected to robotics hardware (sensors, actuators) alongside ROS (Robot Operating System).

UpCom features a multi-tabbed interface with several built-in tabs for robotics software development. Soon you will be able to add user created tabs as well.

The built-in tabs currently available are: Editor, Console, and Camera-feed viewer. Additionally, UpCom features a file explorer to easily view your development workspace with drag-and-drop file management.

Workflow

By default, UpCom will watch a folder called UpRoot which is created on its first launch unless you specify another workspace path. The explorer will show the file system which can be manipulated from within the browser. You can also quickly open files by dragging and dropping files into the editor.

Controls

Helpful control buttons allow you to clean and build your workspace with a single click. Another lets you run ROS launch files and nodes through a simple menu. A share button currently integrated with Git can automatically commit your unstaged package changes and push them to the preconfigured remote repository.

Console

The UpCom console is a projection of your own terminal window into the browser. Having a console integrated into the UpCom IDE means less switching between windows during development.

Tidy

Have more than one project? Easily organize them by adding and removing workspaces on the fly.
