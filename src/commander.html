UpDroid Commander

Robotics Development Simplified

UpCom 0.4.0 is currently in alpha and available for testing. In order to run UpCom you need ROS (Robot Operating System). You can install ROS here. UpCom is supported on Ubuntu 14.04 x86_64 and was tested on Google Chrome version 42.0.2311.90. Happy Programming.

Install Alpha
wget -qO - http://packages.updroid.com/Release.key | sudo apt-key add - sudo sh -c 'echo "deb http://packages.updroid.com/ubuntu/ trusty main" > /etc/apt/sources.list.d/updroid.list'

In order to upgrade commander run: apt-get install --only-upgrade cmdr

Current Features

  • Tabbed components
  • Integrated console, text editor, file explorer and camera stream
  • Build workspace, run nodes, and run version control by pushing a button

Near Future
