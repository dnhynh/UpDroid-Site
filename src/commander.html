UpDroid Commander

Robotics Development Simplified

UpDroid Commander (UpCom) is a browser-based IDE for robotics software development. It can be installed on an Ubuntu 14.04 x86_64 PC that is connected to robotics hardware (sensors, actuators) alongside ROS (Robot Operating System). You can install ROS here.


UpCom is on version 0.4.1. It was tested on Google Chrome version 42.0.2311.90 and Chromium 42.0.2311.90. As UpCom is currently in its Alpha phase, please expect issues and help make us aware of them through it's feedback button. Happy Progamming.

Install Alpha

In order to install UpCom run the following commands in your console:


    $ wget -qO - http://packages.updroid.com/Release.key | sudo apt-key add -   
    
    $ sudo sh -c 'echo "deb http://packages.updroid.com/ubuntu/ trusty main" 
    > /etc/apt/sources.list.d/updroid.list'
    $ sudo apt-get update
    $ sudo apt-get install cmdr


In order to update run: 

    # Expect updates to be released weekly, more or less. 
    $ sudo apt-get update
    $ sudo apt-get install --only-upgrade cmdr   
    
Usage

    # Set the ROS source folder
    $ . /opt/ros/indigo/setup.bash
    # Start UpCom
    $ cmdr gui
    
To access UpCom: tune browser to localhost:12060

    # Because UpCom is still in alpha, it currently doesn't have any built in

UpDroid Commander (UpCom) is a browser based IDE for robotics software development. It features a multi-tabbed interface with several built-in tabs for robotics software development. Soon you will be able to add user created tabs as well. The built-in tabs currently available are: Editor, Console, and Camera-feed viewer. Additionally, UpCom features a file explorer to easily view your development workspace with drag-and-drop file management.

By default, UpCom will watch a folder called UpRoot which is created on its first launch unless you specify another workspace path. The explorer will show the file system which can be manipulated from within the browser. You can also quickly open files by dragging an dropping files into the editor.

The UpCom console is a projection of your own terminal window into the browser. Having a console integrated into the UpCom IDE means less switching between windows during development.

Helpful control buttons allow you to clean and build your workspace with a single click. Another lets you run ROS launch files and nodes through a simple menu. A share button currently integrated with Git can automatically commit your unstaged package changes and push them to the preconfigured remote repository.
