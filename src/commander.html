UpDroid Commander

Commander was created with the intent of making robotics development less painful. ROS is powerful but using it can be a hassle. Scattered workspaces, multiple terminals, complex scripts, and a complicated install to boot. Commander is an IDE which takes advantage of the power of ROS but abstracts out all that nonsense.

Current Features

  • Tabbed components
  • Integrated console, text editor, file explorer and camera stream
  • Build workspace, run nodes, and run version control by pushing a button

Near Future

  • Multiple Workspaces
  • Share code with other users
