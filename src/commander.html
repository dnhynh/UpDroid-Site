UpDroid Commander

UpCom was created with the intent of making robotics development less painful. ROS is powerful but using it can be a hassle. Scattered workspaces, multiple terminals, complex scripts, and a complicated install to boot. UpCom is an IDE which takes advantage of the power of ROS but abstracts out all that nonsense.

Current Features

  • Tabbed components
  • Integrated console, text editor, file explorer and camera stream
  • Build workspace, run nodes, and run version control by pushing a button

Near Future

  • Multiple Workspaces
  • Share code with other users
  • 32 bit support

Download via command line

The above button automatically downloads this following script. However if you want to install manually, run the following commands in your terminal


    sudo apt-get update
    sudo apt-get install apt-transport-https

    # Get the Google Linux package signing key and set up the stable repository.

    curl --silent https://dl-ssl.google.com/linux/linux_signing_key.pub | sudo tee apt-key add -'
    sudo curl --silent https://storage.googleapis.com/download.dartlang.org/linux/debian/dart_stable.list > /etc/apt/sources.list.d/dart_stable.list    

    # Get the ROS package signing key and set up the stable repository.

    sudo apt-key adv --keyserver hkp://pool.sks-keyservers.net --recv-key 0xB01FA116
    sudo echo "deb http://packages.ros.org/ros/ubuntu trusty main" > /etc/apt/sources.list.d/ros-latest.list

    # Get the UpDroid package signing key and set up the dev repository.

    curl --silent http://packages.updroid.com/packages.updroid.com-keyring.key | sudo tee apt-key add -
    sudo echo "deb http://packages.updroid.com/ubuntu/ trusty main" > /etc/apt/sources.list.d/updroid.list

    sudo apt-get update
    sudo apt-get install cmdr
