UpDroid Commander

Robotics Development Simplified

UpCom 0.4.0 is currently in alpha and available for testing. In order to run UpCom you need ROS (Robot Operating System). You can install ROS here. UpCom is supported on Ubuntu 14.04 x86_64 and was tested on Google Chrome version 42.0.2311.90. If you have any comments on how to improve UpCom, you can use the built in feedback tool. Any thoughts you have are always appreciated. Happy Programming.

Install Alpha

In order to install UpCom run the following commands in your console:


    $ wget -qO - http://packages.updroid.com/Release.key | sudo apt-key add -   
    
    $ sudo sh -c 'echo "deb http://packages.updroid.com/ubuntu/ trusty main" 
    > /etc/apt/sources.list.d/updroid.list'
    $ sudo apt-get update
    $ sudo apt-get install cmdr


In order to update run: 

    $ apt-get install --only-upgrade cmdr   
    
Usage

    # Set the ROS source folder
    $ . /opt/ros/indigo/setup.bash
    # Start UpCom
    $ cmdr gui

    
To access UpCom: tune browser to localhost:12060

    # Because UpCom is still in alpha, it currently doesn't have any built in 
    security tools. As such, please run UpCom from within a closed network.
